What this library is for

Google reviewers and normal visitors usually ask the same question in different words: does this site add enough value beyond a button and a score? This library exists to answer that clearly. The goal is to document measurement logic, explain score limits, publish comparison rules, and show the editorial standards behind the main tool.

None of these pages promise a universal ranking system. They focus on repeatable comparison under matching conditions, because that is the most honest use of a browser-based CPS test.

What is included

  • Methodology notes tied to the live NEONCPS input logic.
  • Original guidance on score interpretation and device differences.
  • Safety and correction standards for an input-heavy utility site.

What is not claimed

  • No medical or diagnostic use.
  • No claim that mouse and touch scores are directly equal.
  • No promise that one run is more meaningful than repeated runs in the same setup.

NEONCPS is deliberately narrow in scope. That is a design choice, not an accident. The site focuses on one utility, then adds the surrounding explanation needed for a user to understand what the number means before ads are ever considered.

NEONCPS keeps these pages small, specific, and tied to the live tool. If a guide falls out of sync with the test logic, the correction path is published on the Editorial Policy and About pages.
