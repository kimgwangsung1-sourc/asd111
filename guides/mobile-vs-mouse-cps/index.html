Mobile versus mouse CPS모바일과 마우스 CPS

A touch score and a mouse score can both be real while still describing different tasks. This guide explains why NEONCPS does not recommend simple mixed-device ranking.

터치 점수와 마우스 점수는 모두 실제 기록일 수 있지만 서로 다른 작업을 설명합니다. 이 문서는 NEONCPS가 기기 혼합 순위를 단순하게 권장하지 않는 이유를 설명합니다.

The same score can mean different things on different hardware같은 점수라도 하드웨어가 다르면 의미가 달라질 수 있습니다

Mouse input and touch input travel through different physical and software paths. The motion is different, the contact surface is different, and the latency profile is different.

마우스 입력과 터치 입력은 물리적·소프트웨어적 경로가 다릅니다. 동작도 다르고, 닿는 표면도 다르고, 지연 특성도 다릅니다.

1. Direct contact versus mediated contact1. 직접 접촉과 간접 접촉의 차이

On a phone or tablet, your finger touches the surface that receives the input. On a mouse, your finger presses a switch while the pointer interaction is mediated through separate hardware. That difference changes how rhythm is produced and how fatigue builds.

휴대폰이나 태블릿에서는 손가락이 입력을 받는 표면에 직접 닿습니다. 마우스에서는 손가락이 스위치를 누르고, 실제 포인터 입력은 다른 하드웨어를 거쳐 전달됩니다. 이 차이는 리듬이 만들어지는 방식과 피로가 쌓이는 방식을 바꿉니다.

2. Latency profiles are different2. 지연 경로가 다릅니다

Touchscreens have their own scan and display pipeline. Mouse input has its own report timing, switch behavior, and desktop event path. The important point is not which one is universally better. The important point is that the path is not the same.

터치스크린은 자체적인 스캔과 표시 경로를 갖고 있습니다. 마우스 입력은 보고 주기, 스위치 동작, 데스크톱 이벤트 경로가 따로 있습니다. 중요한 것은 어느 쪽이 항상 더 낫다는 결론이 아니라, 경로 자체가 같지 않다는 점입니다.

4. Practical advice for mobile4. 모바일 실전 팁

  • Stabilize the device first.
  • 먼저 기기를 안정적으로 고정하세요.
  • Use the same finger across repeated attempts.
  • 반복 측정에서는 같은 손가락을 쓰세요.
  • Keep finger position consistent.
  • 손가락 위치를 일정하게 유지하세요.
  • 5-second or 10-second runs can be easier to compare repeatedly.
  • 반복 비교를 쉽게 하려면 5초나 10초 모드가 더 편할 수 있습니다.

5. Practical advice for mouse5. 마우스 실전 팁

  • Keep the same mouse and browser.
  • 같은 마우스와 브라우저를 유지하세요.
  • Use a stable hand position.
  • 손 위치를 안정적으로 유지하세요.
  • Do not over-read tiny differences across switch types or settings.
  • 스위치나 설정 차이에 따른 작은 수치 차이를 과하게 해석하지 마세요.
  • Judge consistency from repeated attempts, not a single peak.
  • 한 번의 최고점보다 반복 기록의 일관성을 보세요.

The honest rule is simple: compare like with like first, then treat cross-device scores as rough context only.
